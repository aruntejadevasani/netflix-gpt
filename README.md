# StreamEase 🎬🚀

    - A Netflix-like entertainment web app built with React, Redux, Tailwind CSS, and Webpack, featuring Firebase authentication and AI-powered movie recommendations via OpenAI.

    - 🔧 Features
        ✅ Browse & Stream Movies 🎥
        ✅ Firebase Authentication 🔑
        ✅ AI-Based Movie Recommendations (Horror, Comedy, Thriller) 🤖
        ✅ Optimized Performance with Webpack ⚡

    - Contributions & feedback are welcome! 🎉
